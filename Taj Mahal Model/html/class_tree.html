void Tree::loadModel ( )

Parser to get the data about vertices and faces from the Wavefront(.obj) file format.

Wavefront(.obj) file opened

Checking if the file exists

<Size of the vertices array

<Line to be scanned

The line is split into parts with space as a delimiter.

If result[0] is "v", the line has data about a vertex.

The coordinates of the vertex are parsed and it is pushed into the vertices vector.

If result[0] is "f", the line has indices of the vertices making a face.

Once the first element is not "f", we break out of the loop.

The elements of result[] are split with '/' delimiter and the elements are stored in vertex1, vertex2 and vertex3.

The first element of vertex 1 denotes the index of the vertex in the vertices array. These indices follow 1-indexing. But our vectors follow 0-indexing. Hence 1 is subtracted.
